A high-performance, cinematic 3D portfolio showcasing advanced AI/ML projects, built with **Next.js**, **Three.js**, and **Framer Motion**.

## 🚀 Featured AI Projects

### 🧠 [Unified Emotion Recognition](https://github.com/harshit-001-it)
*   **Tech**: Python, CNN (TensorFlow/Keras), OpenCV.
*   **Impact**: Real-time sentiment analysis engine trained on unified datasets (FER 2013, AffectNet, Raf-DB, CK+). Optimized with a 1024-node dense layer for low-latency inference.

### 📝 [AI Text Analyzer CLI](https://github.com/harshit-001-it)
*   **Tech**: Python, LLM Pipelines, JSON Schema, Pydantic.
*   **Impact**: A production-grade CLI tool for structured text intelligence. Features complex summarization, classification, and robust error handling with machine-readable outputs.

### 📰 [Fake News Detection](https://github.com/harshit-001-it)
*   **Tech**: Python, NLP, Scikit-learn.
*   **Impact**: Linguistic pattern analysis for misinformation detection using TF-IDF and advanced feature engineering.

## 🛠️ Technical Stack

- **Framework**: Next.js 14+ (App Router)
- **3D Graphics**: Three.js & React Three Fiber
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS & Glassmorphism
- **AI/ML**: Python, TensorFlow, OpenCV, Scikit-Learn
